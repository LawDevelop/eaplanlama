# Logo Kurulum Talimatları

## Adımlar:

1. Gönderdiğiniz logo resmini kaydedin
2. Dosya adını `logo.png` olarak değiştirin
3. `public` klasörüne kopyalayın

Dosya yolu şöyle olmalı:
```
public/logo.png
```

Logo otomatik olarak header'da "Av. Emre Arslan" yazısının yanında görünecektir.

## Logo Özellikleri:
- Format: PNG (şeffaf arka plan önerilir)
- Boyut: 36x36 piksel (otomatik olarak ölçeklendirilir)
- Konum: Header sol üst köşe
